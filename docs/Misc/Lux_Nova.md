# Lux Nova Studios

**Lux Nova Studios** is an independent game studio, which includes multiple members of the Drehmal dev team (1). Its existence was formally announced on December 31, 2025, in a [blog post](https://www.drehmal.net/post/about-drehmal-2-3-and-beyond) which also included information about the [[Drehmal v23|Drehmal 2.3 update]].
{ .annotate }

1. Confirmed to include Drehmal co-lead DomeMaster and Drehmal builder/modeler/animator MoutonDemocrate.

As of now, Lux Nova has no released projects, but has been developing a game since mid-2025 with the intention of releasing a playable demo in early 2026. This project is in no way related to Drehmal, and was revealed to have the initials "DTGI".
